1- extract topic and info from video or blog to see what the user is studying
2- give him/her exam about the studied topic and keep him locked untill he finish studying and solving the exam
3- update the user data by adding the new topic if its new or updating the topic mastery or progress etc based on the topic depth and content and users score in the test 
4- relate similar topics together if they are under the same field ie binary search and dynamic programming under algorithms etc and have a somehow tree relationship between topics
5- give an overview of the user mastery over specific field like machine learning based on his score in the related topics under this field
6- recommend topic to study or focus at given the user topics data
7- (optional) agent to create cv for the user based on his data
